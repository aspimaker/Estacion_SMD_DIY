# Estacion_SMD_DIY
Base de soldadura SMD. Actualmente en desarrollo. No lo construyas todavía.

Utilizando un Arduino Nano, incluye el hardware necesario para calentar y controlar la temperatura usando una NTC y un SSR.
Incluye una pantalla OLED de 0.96", un enconder para moverse por el menú, algunos led y pulsadores.

Está diseñada para utilizar una tira led RGB los estados: frio, calentando, temperatura alcanzada, enfriando...
Puede que lo cambie por una tira WS2812B para simpliicar el diseño y la construcción.

Se podrán crear curvas de temperatura..

Agradezco tu colaboración para ayudarme a con posibles mejoras, corrección de errores, etc...

Gracias y un saludo.

#DIY_SMD_Station
SMD soldering station. Currently under development. Do not build it yet.

Using an Arduino Nano, it includes the necessary hardware for heating and temperature control using an NTC thermistor and an SSR.
It includes a 0.96" OLED screen, an encoder for navigating the menu, some LEDs, and pushbuttons.

It is designed to use an RGB LED strip with the following states: cold, heating, temperature reached, cooling, etc. I may change it to a WS2812B strip to simplify the design and construction.

Temperature curves can be created.

I appreciate your collaboration in helping me with possible improvements, bug fixes, etc.

Thank you and best regards.
